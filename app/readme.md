The Advisor Guide is a dedicated portal where you can access a library of video tutorials designed to help you better serve your clients.

1. Getting Started
Advisor Portal Welcome: Upon visiting the advisor section, you are greeted with a clear welcome message and a prompt to Watch Video Tutorials.

Video Library: You can browse the "Video Tutorials & Insights" gallery, which features specialized guides on document security and platform navigation.

Video Details: Each video includes a title, a brief description of the content, and the publication date.

2. Learning Paths
Sequential Learning: To ensure a structured learning experience, the platform uses an "Up Next" feature.

Continue Learning: After finishing a tutorial, you can easily click Continue to move to the next relevant video in the sequence.

Progress Tracking: The main gallery organizes videos numerically (e.g., #1, #2, #3), making it easy to see the intended viewing order.

The platform includes a secure backend specifically for administrators to manage the content displayed to advisors.

1. Admin Authentication
Sign In: Admins access the management dashboard through a secure login page at admin@safebox.life.

Goal: This portal allows you to upload videos, organize the gallery, and build comprehensive guides for your audience.

2. Video Gallery Dashboard
The Video Gallery provides an at-a-glance overview of your current content:

Total Videos: Displays the current count of uploaded tutorials (e.g., 7 videos).

Sequence Status: Shows how many videos are currently linked in a sequential learning path.

Gallery Management: Admins can view the full list of videos, check which have a "Has Next" link, and use quick actions to view, edit, or delete content.

3. Adding and Organizing Content
The "Add New Video" interface is designed for building step-by-step learning paths:

Required Info: You can add a catchy Video Title, the YouTube URL, and a detailed Description.

Creating the Path: Use the "Next Video in Sequence" dropdown to link videos together, creating a seamless journey for the advisor.

Publication: Once finalized, clicking Publish Video makes it live in the advisor gallery.
